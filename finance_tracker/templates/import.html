Import Transactions from CSV

Bulk upload transactions from a CSV file downloaded from your bank.

CSV Format Instructions

Please ensure your CSV file has **6 columns** in the following order, with a header row:

Date,Description,Amount,Account Name,Account Type,Category
2025-06-15,Grocery Store,-75.50,SBI,Savings,Groceries
2025-06-15,Monthly Salary,5000.00,SBI,Savings,Salary
2025-06-16,Credit Card Bill,-250.00,SBI Credit Card,Credit Card,Bills
  • Date: Must be in YYYY-MM-DD format.
  • Description: A brief description of the transaction.
  • Amount: Use negative numbers for expenses (e.g., -75.50) and positive numbers for income.
  • Account Name: The name of the account (e.g., "SBI", "SBI Credit Card").
  • Account Type: The type of the account (e.g., "Savings", "Credit Card").
    The combination of Account Name and Account Type must exactly match an account you have already created.
  • Category: The name of the category. If the category does not exist, it will be created for you automatically.
